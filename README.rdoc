= Scanty, a really small blog. forked from scanty of adamwiggins

== Overview

Software for my blog, to be exact:
http://subbarao.me

It is not a blogging engine, but it's small and easy to modify, so it could be
the starting point for your blog, too.

== Features

* Posts (shock!)
* Tags
* Markdown (via Maruku)
* Atom feed
* Comments via Disqus
* Web framework = Sinatra
* ORM = postgres
* Syntaxhighlighter(javascript library)
